## Run

Use the package manager [yarn](https://yarnpkg.com/en/) to install the necessary libs and run the project.

```bash
yarn package-clean
```

```bash
yarn start
```

Use the [Jest](https://jestjs.io/) to run unit tests and generate coverage report.
